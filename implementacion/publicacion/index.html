Publicación de Teselas Vectoriales


Para la publicación de las teselas vectoriales generadas con Tippecanoe se utiliza un servidor web con la siguiente configuración:

  • Sistema operativo: Alpine-Docker
  • Procesador: Intel de 2 núcleos
  • Memoria RAM: 4Gb
  • Almacenamiento: Disco duro sólido de 50Gb
  • Servidor web: NGINX


La aplicación que se ha programado para servir las teselas vectoriales se ha desarrollado en NodeJS por su rendimiento y sencillez. Sin embargo podría programarse en cualquier otro lenguaje de programación: Java, Python, PHP, etc.


Un ejemplo del desarrollo en NodeJS sería:

const express = require('express');
const app = express();
const port = process.env.port || 3050;

app.get('/vt-igo/:z/:x/:y.pbf');

const db = new sqlite3.cached.Database(mbFile, sqlite3.OPEN_READONLY);
const sql = 'SELECT tile_data' +
    ' FROM tiles' +
    ' WHERE zoom_level=' + zoom +
    ' AND tile_column=' + column +
    ' AND tile_row=' + row;

db.get(sql, function (err, row) {
    if (row) {
        return fn(row.tile_data);
        if (row.tile_data) {
            res.writeHead(200, {
                'Content-Type': 'application/x-protobuf',
                'Cache-Control': 'public, max-age=2592000'
            });
        } else {
            res.status(404).send('Tile not found');
        }
    } else {
        return fn(false);
    }
});

El proceso consiste en realizar una consulta a la base de datos SQLite, obtener el elemento Tile PBF y devolverlo al navegador.
