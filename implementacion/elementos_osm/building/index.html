Elementos OSM: building


building

La etiqueta building se utiliza para marcar un objeto dado como un edificio. El uso más básico es building=yes, pero el valor se puede usar para clasificar el tipo de construcción.

Nombres: building_wayrel
Filtros: Tag building exist
Geometrías: multilinestring, multipolygon
