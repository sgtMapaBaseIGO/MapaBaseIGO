Elementos OSM: leisure


leisure_nature_reserve

Áreas protegidas de importancia para la vida silvestre, flora, fauna o características de interés geológico u otro interés especial, que se reservan y administran para la conservación y para brindar oportunidades especiales de estudio o investigación. En algunos países las reservas naturales son creadas por la Administración o por entidades privadas, como organizaciones benéficas e instituciones de investigación.

Nombres: leisure_nature_reserve_wayrel
Filtros: leisure=nature_reserve
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
boundary string Límite
(aboriginal_lands, administrative, historic, maritime, marker, national_park, political, postal_code, protected_area, religious_administration, user defined, ...) Tipología
leisure string Recreo

name string Nombre
name:es string Nombre en español

(wood, tree_row, tree, scrub, heath, moor, grassland, fell, bare_rock, scree, shingle, sand, mud, water, wetland, glacier, bay, cape, peninsula, beach, strait, coastline, spring, hot_spring, geyser, reef, peak, hill, volcano, valley, river_terrace, ridge, arete, cliff, saddle, rock, stone, cave_entrance, sinkhole, user defined, ...) Tipología
protect_class string Clase de protección

leisure_park

Parques, espacios abiertos para uso recreativo, generalmente diseñados y en estado seminatural con áreas cubiertas de hierba, árboles y arbustos.

Nombres: leisure_park_wayrel
Filtros: leisure=park
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
leisure string Recreo
park Parque
name string Nombre
name:es string Nombre en español
