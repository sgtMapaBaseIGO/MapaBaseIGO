Elementos OSM: water


water_canal

Canal.

Nombres: water_canal_wayrel
Filtros: water=canal
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
name string Nombre
name:es string Nombre en español
service string Servicio
(transportation, water_power, irrigation) Tipo de servicio
water string Agua
canal Canal

water_lagoon

Laguna. Cuerpo de agua de mar poco profunda o agua salobre separada del mar por algún tipo de barrera. Es mejor especificar la salinidad del agua con sal = sí / no.

Nombres: water_lagoon_wayrel
Filtros: water=lagoon
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
intermittent string Intermitente
yes Sí
name string Nombre
name:es string Nombre en español
water string Agua
lagoon Albufera

water_lake

Lago. Cuerpo natural de agua de tamaño considerable, localizado en una cuenca rodeada de tierra.

Nombres: water_lake_wayrel
Filtros: water=lake
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
intermittent string Intermitente
yes Sí
name string Nombre
name:es string Nombre en español
sqkm int Área en km2
water string Agua
lake Albufera

water_oxbow

Lago. Cuerpo natural de agua de tamaño considerable, localizado en una cuenca rodeada de tierra.

Nombres: water_oxbow_wayrel
Filtros: water=oxbow
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
name string Nombre
name:es string Nombre en español
water string Agua
oxbow Meandro residual

water_reservoir

Embalse o lago artificial para almacenar agua. Generalmente formado por una presa sobre un curso de agua natural.

Nombres: water_reservoir_wayrel
Filtros: water=reservoir
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
name string Nombre
name:es string Nombre en español
water string Agua
reservoir Embalse

water_river

Río. Área cubierta de agua de un río.

Nombres: water_river_wayrel
Filtros: water=river
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
name string Nombre
name:es string Nombre en español
water string Agua
river Río
