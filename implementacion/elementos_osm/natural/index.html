Elementos OSM: natural


natural_water

Masa de agua contiental, como un lago o estanque. Para los lagos se añade además la etiqueta water=lake.

Nombres: natural_water_wayrel
Filtros: natural=water
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
intermittent string Intermitente (no contiene agua de forma permanente)
yes Sí
name string Nombre
name:es string Nombre en español
natural string Naturaleza de los elementos
water Cuerpo de agua
water string Naturaleza del agua
basin Balsa
canal Canal
fish_pass Gradas para peces

lake Lago
lock Esclusa
