Elementos OSM: highway


highway_living_street

La etiqueta highway=living_street se usa para etiquetar calles u otras implementaciones de espacio compartido. Este tipo de vías tienen límites de velocidad más bajos y reglas especiales de tráfico y estacionamiento en comparación con las calles urbanas. En estas calles los peatones tienen el mismo derecho de paso que otros usuarios de la vía. Los límites de velocidad implícitos establecidos en la legislación varían desde la "velocidad de marcha" hasta los 20 km/h en algunos países. El límite de velocidad se señaliza de forma explícita.

Nombres: highway_living_street_wayrel
Filtros: highway=living_street
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
bridge string Puente
yes Sí
highway string Vía
living_street Vía urbana residencial
name string Nombre
name:es string Nombre en español
tunnel string Túnel
yes Sí

highway_motorway

Autopistas y autovías. La etiqueta highway=motorway se utiliza para identificar las carreteras de mayor rendimiento dentro de un territorio. Típicamente tienen al menos dos carriles en cada dirección de tráfico con cierto grado de separación. Disponen de vías de acceso para fines de servicio y accesos a áreas de descanso.

Nombres: highway_motorway_wayrel
Filtros: highway=motorway,highway=motorway_link
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
bridge string Puente
yes Sí
highway string Vía
motorway Autopista/Autovía
motorway_link Enlace Autopista/Autovía
name:es string Nombre en español
ref string Referencia
tunnel string Túnel
yes Sí

highway_pedestrian

Vías peatonales. Una highway=pedestrian es una carretera o un área de uso exclusivo para peatones, en las que se puede autorizar el tráfico de algunos vehículos (por ejemplo, emergencia, taxi, entrega, ...). Por lo general, se encuentran en áreas comerciales, centros urbanos, lugares con atracciones turísticas y áreas recreativas/cívicas donde se existen grandes extensiones de superficie dura para que los peatones caminen.

Nombres: highway_pedestrian_wayrel
Filtros: highway=pedestrian
Geometrías: multilinestring, multipolygon
Atributos:

yes Sí

highway_primary

Carreteras importantes que unen ciudades grandes. En países desarrollados normalmente con 2 carriles, si bien en áreas con peor infraestructura la calidad de las carreteras puede ser mucho peor. Normalmente no cuentan con una barrera central de separación entre ambas direcciones.

highway_secondary

Carreteras que no forman parte de una ruta importante pero sirven de enlace en la red nacional de rutas. En algunos países tienen 2 carriles y el tráfico en ambas direcciones suele estar separado por una línea central en la carretera. En zonas con peor infraestructura la calidad de las carreteras puede ser mucho peor.

highway_tertiary

Las carreteras terciarias son carreteras que conectan asentamientos más pequeños y dentro de grandes asentamientos conectan centros locales. En términos de la red de transporte, las carreteras "terciarias" de OpenStreetMap comúnmente también conectan las calles menores con las carreteras más importantes.
