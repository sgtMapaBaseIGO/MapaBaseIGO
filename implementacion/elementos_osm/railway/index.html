Elementos OSM: railway


railway_platform

Plataformas ferroviarias. Andenes.

Nombres: railway_platform_wayrel
Filtros: railway=platform
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
name string Nombre
name:es string Nombre en español
subway string Suburbano/Metro
yes Sí
railway string Tipo de vía férrea
platform Andén
wheelchair string Accesible para sillas de ruedas
yes Sí

railway_rail

Carriles de una vía de ancho estándar. Se utiliza para trenes de pasajeros o de carga. Es la clasificaciín ferroviaria más utilizada.

Nombres: railway_rail_wayrel
Filtros: railway=rail
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
electrified string Tipo de electrificación
contact_line Rieles o calbes aéreos
no No
rail Raíl
yes Sí
highspeed string Alta velocidad
yes Sí
name string Nombre
name:es string Nombre en español
railway string Tipo de vía férrea
rail Tren de pasajeros/carga

railway_station

Estaciones de tren. Lugares donde los clientes pueden acceder a servicios ferroviarios o donde se cargan y descargan mercancías.

Nombres: railway_station_wayrel
Filtros: railway=station
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
name string Nombre
name:es string Nombre en español
public_transport string Transporte público
platform Plataforma
station Estación
stop_area Zona de parada
stop_position Punto de parada
railway string Tipo de vía férrea
station Estación
wheelchair string Accesible para sillas de ruedas
yes Sí

railway_tram

Vías de tranvía. Se utiliza para vías que se usan principalmente o exclusivamente para tranvías, o donde las vías de tranvía se colocan dentro de una carretera normal abierta a todo el tráfico.

Nombres: railway_tram_wayrel
Filtros: railway=tram
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
electrified string Tipo de electrificación
contact_line Rieles o calbes aéreos
no No
rail Raíl
yes Sí
name string Nombre
name:es string Nombre en español
railway string Tipo de vía férrea
rail Tren de pasajeros/carga
service string Servicio
(crossover, siding, spur, yard) Tipo de servicio
