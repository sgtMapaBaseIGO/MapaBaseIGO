Elementos OSM: place


place_city

Puntos de referencia para etiquetado de ciudades. La etiqueta place=city se utiliza para identificar el asentamiento o asentamientos más grandes dentro de un territorio, incluidas las capitales nacionales, estatales y provinciales y otras con urbaciones importantes.

Nombres: place_city_node
Filtros: place=city
Geometrías: point
Atributos:

Atributo Tipo Valor Descripción
admin_level int Nivel administrativo
(1-12) Jerarquía del nivel administrativo
capital string Capital
yes Capital de un país
(1-12) Orden de jerarquía de la capitalidad
name string Nombre
name:es string Nombre en español
place string Tipo de localización
city Ciudad
population int Población

place_town

Puntos de referencia para etiquetado de centros urbanos. Se utiliza la etiqueta place = town para identificar un centro urbano importante que sea más grande que un place = village, más pequeño que un place = city, y no un place = suburbio.

Nombres: place_town_node
Filtros: place=town
Geometrías: point
Atributos:

Atributo Tipo Valor Descripción
name string Nombre
name:es string Nombre en español
place string Lugar
town Centro urbano
population int Población
