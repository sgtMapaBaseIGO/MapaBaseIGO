Elementos OSM: boundary


boundary_national_park

Parques nacionales. En OSM se definen como áreas de tierra relativamente grandes declaradas por un gobierno. Deben reservarse para la recreación y el disfrute humanos, así como para la protección del medio ambiente natural y/o patrimonio cultural de una zona.

Nombres: boundary_national_park_wayrel
Filtros: boundary=national_park
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
boundary string Límite
national_park Parque Nacional
name string Nombre
name:es string Nombre en español
protection_object string (aborigine, nature, habitat, art, econonic, politic, military, historic, water, soil, air, climat, noise, floodwater, rockslide, retention, ...) Objeto de protección
protection_title string (Wildlife sanctuary, Zapovidnyk, Nationalpark, Special Protected Area, World Heritage Convention, Aesthetic Forest, Heritage River, Indian Reserve, ...) Tipo de protección

boundary_protected_area

Áreas protegidas. En OSM se utiliza para marcar los límites de las áreas protegidas, incluidos los parques nacionales, las áreas de protección marina, los sitios de patrimonio, áreas silvestres, áreas indígenas y otras similares.

Nombres: boundary_protected_area_wayrel
Filtros: boundary=protected_area
Geometrías: multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
boundary string Límite
protected_area Área protegida
name string Nombre
name:es string Nombre en español
