Elementos OSM: aeroway


aeroway_aerodrome

Aeródromos. La tecla aeroway=aerodrome se utiliza para representar todos los detalles principales de un aeródromo. Preferiblemente para el punto de referencia del aeródromo (ARP), aunque también puede estar asociado con el área alrededor de su perímetro.

Nombres: aeroway_aerodrome_node, aeroway_aerodrome_wayrel
Filtros: aeroway=aerodrome
Geometrías: point, multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
aeroway string Pista aérea
aerodrome Aeródromo
aerodrome:type string Tipo de aeropuerto
airfield Campo de aviación
gliding Vuelo sin motor
international Internacional
military Militar
private Privado
regional Regional
name string Nombre
name:es string Nombre en español

aeroway_heliport

Helipuertos y pistas de aterrizaje para helicópteros. Un Helipuerto es un aeródromo especial para el uso exclusivo de helicópteros. Es diferente de aeroway=aerodrome ya que no sirve para aeronaves de ala fija. Generalmente tiene una o más pistas de aterrizaje de forma circular señalizadas con una H central visible desde el aire.

Nombres: aeroway_heliport_node, aeroway_heliport_wayrel
Filtros: aeroway=heliport, aeroway=helipad
Geometrías: point, multilinestring, multipolygon
Atributos:

Atributo Tipo Valor Descripción
aeroway string Pista aérea
heliport Helipuerto
helipad Punto de aterrizaje de helicópteros
name string Nombre
name:es string Nombre en español
