Simbolización de Tiles Vectoriales


Implementación para MapboxGL

La implementación del estilo para MapboxGL se realiza mediante un archivo JSON donde se definen la simbología de cada uno de los elementos y los niveles de zoom en los que aparecen. Se puede utilizar la herramienta Maputnik como editor online de estilos.

Se proponen dos estilos, accesibles desde las siguientes URLs:

  • Estilo claro: https://vts.larioja.org/style/mapa-base-igo-v1.json
  • Estilo oscuro: https://vts.larioja.org/style/mapa-base-igo-oscuro-v1.json

Una muestra del contenido del archivo JSON de estilo podría ser:

{
  "version": 8,
  "name": "mapabaseIGO",
  "metadata": {},
  "sources": {
    "vts-mapabaseIGO": {
      "type": "vector",
      "tiles": [
        "https://vts.larioja.org/ts/vt/igo/{z}/{x}/{y}.pbf"
      ],
      "minZoom": 0,
      "maxZoom": 22
    }
  },
  "sprite": "https://vts.larioja.org/sprite/sprite",
  "glyphs": "https://vts.larioja.org/font/{fontstack}/{range}.pbf",
  "layers": [
    {
      "id": "fondo",
      "type": "background",
      "minzoom": 0,
      "maxzoom": 22,
      "layout": {
        "visibility": "visible"
      },
      "paint": {
        "background-color": "rgba(201, 201, 201, 1)"
      }
    },
    {
      "id": "mfe50",
      "type": "fill",
      "metadata": {
        "maputnik:comment": "6-22"
      },
      "source": "vts-mapabaseIGO",
      "source-layer": "mfe50_igo",
      "minzoom": 6,
      "maxzoom": 22,
      "filter": [
        "all",
        [
          "!=",
          "clase",
          "artificial"
        ],
        [
          "!=",
          "clase",
          "Artificial"
        ],
        [
          "!=",
          "clase",
          "desarbolado"
        ],
        [
          "!=",
          "clase",
          "Desarbolado"
        ],
        [
          "!=",
          "clase",
          "cultivos"
        ],
        [
          "!=",
          "clase",
          "Cultivos"
        ]
      ],
      "layout": {
        "visibility": "visible"
      },
      "paint": {
        "fill-color": "rgba(219, 234, 210, 1)"
      }
    }]
}

Implementación para HERE Maps

Para visualizar los tiles vectoriales mediante la librería harp.gl de HERE Maps hemos de configurar también un fichero JSON donde definir los estilos y niveles de zoom.

Un ejemplo de implementación sería:

{
    "sky": {
        "type": "cubemap",
        "positiveX": "/examples/harp_gl/img/Sky_px.png",
        "negativeX": "/examples/harp_gl/img/Sky_nx.png",
        "positiveY": "/examples/harp_gl/img/Sky_py.png",
        "negativeY": "/examples/harp_gl/img/Sky_ny.png",
        "positiveZ": "/examples/harp_gl/img/Sky_pz.png",
        "negativeZ": "/examples/harp_gl/img/Sky_nz.png"
    },
    "fog": {
        "color": "#300A00",
        "startRatio": 0.9
    },
    "clearColor": "#224646",
    "lights": [{
            "type": "ambient",
            "color": "#FFFFFF",
            "name": "ambientLight",
            "intensity": 0.9
        }, {
            "type": "directional",
            "color": "#CCCBBB",
            "name": "light1",
            "intensity": 0.8,
            "direction": {
                "x": 1,
                "y": 5,
                "z": 0.5
            }
        }, {
            "type": "directional",
            "color": "#F4DB9C",
            "name": "light2",
            "intensity": 0.8,
            "direction": {
                "x": -1,
                "y": -3,
                "z": 1
            }
        }
    ],
    "styles": {
        "mapabase": [{
                "description": "tierra-firme",
                "when": [
                    "all",
                    ["==", ["get", "$layer"], "tierra_firme_igo"],
                    ["==", ["get", "$geometryType"], "polygon"]
                ],
                "technique": "fill",
                "attr": {
                    "color": "#0A1024"
                },
                "renderOrder": 1
            }, {
                "description": "hidrografia_pol",
                "when": [
                    "all",
                    ["==", ["get", "$layer"], "hidrografia_igo"],
                    ["==", ["get", "$geometryType"], "polygon"]
                ],
                "technique": "fill",
                "attr": {
                    "color": "#224646"
                },
                "renderOrder": 5
            }, {
                "description": "hidrografia_lin",
                "when": [
                    "all",
                    ["==", ["get", "$layer"], "hidrografia_igo"],
                    ["==", ["get", "$geometryType"], "line"]
                ],
                "technique": "solid-line",
                "attr": {
                    "color": "#5b7ca6",
                    "lineWidth": 5,
                    "fadeNear": 0.8,
                    "fadeFar": 0.9,
                    "clipping": false
                },
                "renderOrder": 6
            }
        ]
    }
}
