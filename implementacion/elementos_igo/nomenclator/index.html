Elementos IGO: Nomenclátor


capital_municipio

Ubicación geográfica y datos básicos de las capitales de los municipios de España, recopilados en el «Nomenclátor Geográfico de Municipios y Entidades de Población» que mantiene el Instituto Geográfico Nacional (IGN) y distribuye el Centro Nacional de Información Geográfica (CNIG) como Información Geográfica de Referencia.

Archivo: capital_municipio_pto
Geometría: punto
Fuente de datos: Nomenclator_Municipios_EntidadesDePoblacion.mdb (IGN/CNIG. IGR Nomenclátor Geográfico de Municipios y Entidades de Población)
Filtrado:

Clasificación Condición
clase: capital_municipio Tabla:MUNICIPIOS (todos los registros)

Atributos:

Atributo Tipo Valor Observaciones
clase string capital_municipio
municipio string MUNICIPIOS:[NOMBRE_ACTUAL]
nombre string MUNICIPIOS:[CAPITAL]
poblacion_capital string MUNICIPIOS:[POBLACION]
provincia string MUNICIPIOS:[PROVINCIA]
