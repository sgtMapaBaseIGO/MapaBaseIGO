Teselas vectoriales ICGC

(BORRADOR/CONTENIDO PROVISIONAL)


El Institut Cartogràfic i Geològic de Catalunya desarrolla distintos proyectos de innovación y prototipaje de productos y servicios en el ámbito de la geoinformación, que publica a través del BETA PORTAL ICGC

Algunos de estos proyectos tienen como objetivo la simbolización cartográfica a partir de teselas vectoriales.

El estilizado se realiza con la librería JavaScript Mapbox GL JS a partir de las teselas vectoriales que son servidas con el software TileServer GL de klokantech.

Para confirgurar los proyectos cartográficso, el ICGC edita distintos ficheros JSON de estilizado, conformes con la especificación de MapBox, así como los ficheros sprite (bitmaps bidimensionales) y glyphs (glifos), que se necesitan para la representación de iconos y textos.

Además de la información que distribuye el ICGC a través del Beta Portal, es posible acceder a los recursos almacenados en la cuenta de GitHub OpenICGC y la página web Recursos ICGC en abierto.

Actualmente el ICGC está desarrollando proyectos para las escalas 1:5.000 y 1:25.000.

A partir de la información que distribuye este organismo, hemos hecho una selección de distintos proyectos de mapas teselados con objeto de dar una idea de la flexibilidad que ofrece la cartografía producida a partir de teselas vectoriales.

Mapas Topográficos de Cataluña 1:5.000


Polit



Este proyecto utiliza los siguientes recursos:

Servicios de teselas vectoriales:

  • bt5m: https://tilemaps.icgc.cat/tileserver/tileserver.php/bt5m_vector/{z}/{x}/{y}.pbf
  • Toponimia: https://tilemaps.icgc.cat/tileserver/tileserver.php/redtopo/{z}/{x}/{y}.pbf
  • catpol: https://tilemaps.icgc.cat/tileserver/tileserver.php/catpol4326/{z}/{x}/{y}.pbf
  • openmaptiles: https://free.tilehosting.com/data/v3.json?key=QLOqi8gs14T1s1yjnRMO

Servicios de teselas ráster:

  • terrain: https://tilemaps.icgc.cat/tileserver/tileserver.php/mapboxTerrain/{z}/{x}/{y}.png

Fichero de estilo: https://tilemaps.icgc.cat/tileserver/styles/polit.json
sprite: https://tilemaps.icgc.cat/tileserver/sprites/maki
glyphs: https://tilemaps.icgc.cat/tileserver/glyphs/{fontstack}/{range}.pbf
