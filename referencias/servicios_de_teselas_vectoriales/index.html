Servicios de Teselas Vectoriales

(BORRADOR/CONTENIDO PROVISIONAL)


Cada día son más numerosas las organizaciones que ofrecen servicios de distribución de información geográfica mediante la tecnología de teselas vectoriales.

Algunos de estos servicios de carácter global, ofrecen la posibilidad de crear mapas de referencia de cualquier lugar del mundo con distintos niveles de detalle.

La mayor parte de ellos generan las teselas que ofertan a partir de la información ofrecida por OpenStreetMap (OSM), utilizando para ello distintos esquemas o modelos de datos.

Recogemos aquí algunos de estos modelos con objeto de que puedan servir de referencia, sin la pretensión de realizar una recopilación completa y exhaustiva de toda la oferta existente


OpenMapTiles


OpenMapTiles es un proyecto de software abierto de Klokan Technologies GmbH, que ofrece la posibilidad de instalar libremente el servidor de teselas estilizadas OpenMapTiles Map Server.

Las teselas producidas por esta aplicación, pueden ser consumidas por visualizadores de mapas JavaScript así como por aplicaciones nativas para dispositivos móviles iOS y Android.

Este software se nutre de los paquetes de teselas vectoriales ofrecidos por OpenMapTiles a través de su servicio de descarga de teselas vectoriales, conforme con la especificación MVT (Mapbox Vector Tile).

El servicio de descarga de teselas vectoriales es de pago, existiendo la posibilidad de descargar la información de forma gratuita en caso de proyectos no comerciales o de software/datos abiertos.

Para facilitar el estilizado de su esquema de datos, openMapTiles distribuye distintas colecciones de ficheros de estilos, cuyo uso es libre.

El esquema de teselas vectoriales de OpenMapTiles, codifica información de OpenStreetMap, Natural Earth y datos geográficos abiertos.

OpenMapTiles ofrece también el software libre Tileserver GL para la distribución de teselas ráster y vectoriales, el software MapTiler desktop para la producción de mapas teselados a partir de información ráster y vectorial, con un uso gratuito limitado; y servicios de alojamiento de mapas personalizados en la nube: MapTiler Cloud, con planes limitados para su uso libre.




Esri


Esri es una compañía de servicios cartográficos creadora del software de análisis y representación cartográfica ArcGIS. Junto con las soluciones geomáticas de escritorio, Esri ofrece una plataforma de trabajo y servicios de mapas online: ArcGIS Online.

Como otras empresas del sector, Esri ha desarrollado en los últimos años, herramientas para la producción, estilizado y distribución de mapas renderizados a partir de teselas vectoriales.

Además de las teselas vectoriales que puede generar un usuario, Esri ha desarrollado distintas colecciones de teselas vectoriales de ámbito global, a partir de las cuales oferta a sus clientes una amplia colección de mapas base, que estos pueden utilizar directamente o reestilizar con objeto de diseñar una propuesta cartográfica personal.

El resultado obtenido puede ser utilizado en soluciones webmapping y aplicaciones para dispositivos móviles.

La estructura y configuración de las entidades que Esri utiliza para la producción sus mapas base, se puede consultar en el documento: Esri Vector Basemap Reference Document (v2)




Mapbox


Mapbox es una empresa que ofrece principalmente servicios cartográficos de pago, que complementa con soluciones de geocodificación directa e inversa de direcciones, routing en función de datos de tráfico, y análisis geoespacial a partir de la librería JavaScript Turf.js.

Mapbox es uno de los actores más influyentes en lo que respecta a teselas vectoriales, ya que esta compañía fue la creadora del estándar abierto para la producción y gestión de teselas vectoriales, denominado Mapbox Vector Tile Specification, conocido como MVT (Mapbox Vector Tile), de uso muy generalizado.

Esta especificación incorpora el método Protobuf (Protocol Buffers) de Google para la serialización estructurada de datos, con objeto de conseguir un almacenamiento y procesado más eficiente.

Esta compañía ofrece a sus clientes varios servicios de distribución de teselas vectoriales de ámbito global, siendo el principal de ellos el denominado "Streets", diseñado a partir de la base de datos geográfica OpenStreetMap (OSM).

Actualmente el esquema de datos estable de esta colección es Mapbox Streets v7, si bien ya está disponible en versión beta el esquema Mapbox Streets v8

Mapbox oferta a sus usuarios otros dos servicios de teselas vectoriales, mediante los que distribuye datos para la renderización del relieve (Mapbox Terrain) y datos actualizados de congestión del tráfico (Mapbox Traffic v1).

Los usuarios de Mapbox disponen de un entorno para la estilización de las teselas vectoriales denominado Mapbox Studio que les ofrece la posibilidad de configurar una simbología de mapas propia.

Aunque los servicios de Mapbox son de pago, existe la posibilidad de utilizarlos gratuitamente de forma limitada, en entornos webmapping y desarrollos SDK para móviles.




Mapbox es una compañía que ha apostado fuertemente por el desarrollo de software abierto, lo que hace posible que muchas de sus aplicaciones, librerías y utilidades puedan ser utilizadas de forma libre.

Entre las muchas aplicaciones que distribuye, resultan particularmente interesantes las siguientes:

  • tippecanoe: Utilidad que ofrece la posibilidad de generar, con un gran rendimiento, colecciones de teselas vectoriales en formatos: GeoJSON, Geobuf y CSV.
  • Mapbox GL JS: Librería JavaScript para visualizar mapas en la web de forma interactiva. Permite aplicar a las teselas vectoriales estilos conforme a la especificación Mapbox Style Specification y presentar sus resultados en entornos WebGL (especificación OpenGL ES 2.0 para gráficos 3D).

Mapbox mantiene en GitHub una amplia oferta de repositorios, desde donde es posible acceder y descargar el código de todos sus desarrollos.

MAPCAT


MAPCAT Es un portal web de geoservicios operado por la compañía húngara GLI Solutions Ltd..

Está página web oferta a sus usuarios las siguientes utilidades:

  • Planificación de rutas para coches, bicicletas, peatones, excursonistas y transporte público.
  • Búsqueda y gestión de puntos de interés (POI).
  • Servicios de mapas a partir de teselas ráster y vectoriales.
  • Codificación/decodificación de localizaciones.

Como otras empresas del sector, ofrece un conjunto de servicios gratuitos para un número acotado de sesiones (una sesión de visualización comprende por término medio 50 peticiones a la API), a partir de las cuales es necesario contratar un plan de uso, que en este caso se subscribe para todo el conjunto de servicios que ofrece la empresa.

El acceso a los servicios y teselas de mapas se realiza a través de una API.

Las teselas vectoriales que proporciona esta compañía, como viene siendo habitual en este tipo de servicios, están construidas a partir de la base de datos geográfica OpenStreetMap.

El esquema de datos que se ofrece en la página incluye información de las capas, atributos y dominios de valores, está correctamente estructurado y según se informa en la misma, se basa en la licencia BSD (Berkeley Software Distribution) del esquema de la compañía OpenMapTiles.
